**Employee Payroll Management System**

How to run : 

Creating SQL : 
  1. Copy the SQL QUERIES from the database folder.
  2. Add it to mysql and run.

Project Run : 
1. Add your SQLusername and Passward in DBConnection.java file.
2. Add mysql/j Connector.jar file into the folder.
3. run javac -cp .:/path_to_project mysql-connector-j-N.N.N.jar *.java.
4. java -cp .:/path_to_project mysql-connector-j-N.N.N.jar PayrollApp.java.
5. Go through all the given cases.
